# Boardcamp API

API REST desenvolvida com **Spring Boot** para gerenciamento de uma locadora de jogos de tabuleiro.

## Tecnologias

* Java 21
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Maven
* JUnit / Mockito

## Funcionalidades

* Cadastro e listagem de **jogos**
* Cadastro e listagem de **clientes**
* Registro e controle de **aluguéis**
